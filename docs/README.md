# AI News Hub / AI 资讯中心

## 项目简介

一个 AI 资讯聚合与消息收录平台，支持 OpenClaw 等 AI 智能体集成，提供丰富的交互动画体验。

## 核心特性

- AI 资讯聚合与展示
- OpenClaw 标准化接入
- 拟人化 3D 交互动画
- RAG 智能问答（规划中）
- AI 智能体编排（规划中）

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | React + TypeScript + Vite |
| 3D 动画 | React Three Fiber + Three.js |
| 状态管理 | Zustand |
| 后端 | FastAPI |
| 数据库 | PostgreSQL + Qdrant |
| AI 集成 | LangChain (预留) |
| 认证 | JWT + OAuth2 |
